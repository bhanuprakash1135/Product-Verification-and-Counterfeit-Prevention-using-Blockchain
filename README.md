Step 1 : Create a folder named "original_barcodes" inside your project directory.This folder will store the generated QR codes.
Step 2 : Open Command Prompt, navigate to your project folder, and run: "python generate_qrcodes.py"
Step 3 : After closing the previous command, open Command Prompt again, go to your project folder, and run: "python Main.py".
Step 4 : Now, open index.html file.
